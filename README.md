
# Pokedex App

## 📌 Descripción
Aplicación que consume una API de Pokémon para mostrar información detallada de cada personaje.

## 🚀 Tecnologías
- Java / Python (según uses)
- API REST

## ⚙️ Funcionalidades
- Búsqueda de Pokémon
- Visualización de datos
- Consumo de API
