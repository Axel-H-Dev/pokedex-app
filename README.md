# Pokedex App

## 📌 Descripción
Aplicación que consume una API de Pokémon para mostrar información detallada de cada personaje. Permite buscar y visualizar datos de manera dinámica.

## 🚀 Tecnologías
- Dart
- API REST
- JSON

## ⚙️ Funcionalidades
- Búsqueda de Pokémon
- Visualización de información
- Consumo de API externa
- Manejo de datos en formato JSON

## 🧠 Aprendizajes
- Consumo de APIs
- Manejo de datos
- Lógica de programación

## ▶️ Cómo ejecutar
1. Clonar el repositorio
2. Ejecutar el proyecto desde el entorno de desarrollo
